What is forkThis?

forkThis is a collaboration tool which allows for an easy way to teach programming to "classes" of any size. From the lecture hall to a small workshop, this learning application connects teachers with students in real time.

Instructors host a session which students can then connect to, through this session instructors are able to create and edit code which all members of the session recieve in real time. This brings the lesson from a powerpoint or whiteboard into the students hands.

Once you have joined a session, at any point you may copy over the current instance locally, where you will be able to edit it at will. Creating a local instance affords the student with the ability to test their ideas and questions as fast as you are able to come up with them.

At any point you are able to save the local instance you have, whether you have just copied over the instructor version, or have made your own changes you would like to remember.
