Or, why we made this application and how you can use it

What is forkThis?

forkThis is a collaboration tool which allows for an easy way to teach programming to "classes" of any size. From the lecture hall to a small workshop, this learning application connects teachers with students in real time.

Instructors host a session which students can then connect to, through this session instructors are able to create and edit code which all members of the session recieve in real time. This brings the lesson from a powerpoint or whiteboard into the students hands.

Once you have joined a session, at any point you may copy over the current instance locally, where you will be able to edit it at will. Creating a local instance affords the student with the ability to test their ideas and questions as fast as you are able to come up with them.

At any point you are able to save the local instance you have, whether you have just copied over the instructor version, or have made your own changes you would like to remember.

How do you use forkThis?

Joining a session:

To get started, join a session from the home page! Just enter the name of the session you are trying to join into the search bar.
Or, alternatively, get the session owner to send you a link to the session.

Using a session:

The left file viewer is the instructor version of the code. This is where the lesson takes place. Student interaction with the instructor code is "limited" to viewing and running. To test your own ideas against this code, or make changes and see how their result, press the Make Instance tab.

In the local instance viewer:
The student may make any changes they wish, and run the code. This viewer is your file and this is where participating with the lesson takes place. The student may now do whatever they wish to the instructor code.

Saving:

You may save any file by writing in a save name and pressing the save file button in the control panel. This will then be openable as a new session in which you are the owner.

Hosting and Owning:

Press the Host a Session button on the home page or in the dashboard. Create a name for the session.

As the owner of a session you control the left panel; and what all students who join your session will see. The owner has the interaction permissions for the left panel that a student has on the right panel.
